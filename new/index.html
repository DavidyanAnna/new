Добро пожаловать

Данный сайт создан с целью помощи первокурсникам ОП "Психология" в процессе адаптации к учебному процессу в НИУ ВШЭ

Кроме того, сайт может быть полезен для абитуриентов, хотя, они и не являются целевой аудиторией данного продукта

Краткое описание страниц

  1. На странице "Основное" содержится наибольшее количество информации. Можно узнать:
    1. про учёбу
    2. про внеучебную деятельность
    3. про цифровые сервисы НИУ ВШЭ
  2. На странице "Полезное" есть контакты учебного офиса и прочих представителей администрации, а также небольшой словарик вышкинских слов
  3. На странице "Фото" можно посмотреть фотографии департамента психологии (для вдохновления!)
  4. По клику на "Сайт ОП Психология" можно попасть на официальную страницу департамента психологии
  5. По клику на "Задать вопрос" можно заполнить форму обратной связи
